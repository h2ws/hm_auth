黑马Portal命令认证工具  
功能： 发送认证请求  

专业使用姿势：
0. 创建links目录，用来放链接脚本 (如果用bxg也推荐放这里)
1. 将links放入path系统环境变量中
2. 在此创建一个auth.bat脚本
3. 写入内容 python run.bat的路径
4. Win + R, 然后输入auth, 回车
5. 认证成功~

简易使用姿势
1. 下载[集成exe](http://https://www.lanzous.com/i2uhpfe)
2. 将auth.exe移动至C:/Windows
3. Win + R, 输入auth, 回车
4. OK~
